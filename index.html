SmartThings OAuth Handler

This page handles OAuth2 authentication for the Stream Deck SmartThings plugin. If you arrived here directly, there's nothing to do!

🎮 For Stream Deck Users

This page will automatically open when you authorize the SmartThings plugin. You'll see your authorization code here to copy back into Stream Deck.

To get started:

  • Install the SmartThings plugin in Stream Deck
  • Add a SmartThings button to your Stream Deck
  • Click "Open Authorization Page"
  • Log in to SmartThings and authorize
  • You'll be redirected back here with your code

👨‍💻 For Developers

This is a simple static OAuth2 callback handler. It extracts the authorization code from the URL query parameters and displays it to the user.

Callback URL: /oauth-callback.html
